进入分支：
git checkout master
上传文件到master：
#1 git add .
#2 git status
#3 git commit -m'解释说明'
#4 git push origin master


拉取文件
git pull origin master


git从master分支拉取新分支
 git checkout master       却换到master分支

2:： git pull                             更新到最新代码

3： git  checkout -b dev       创建新分支并切换到该分支

4：git push origin dev         推送新分支到远程仓库

5：git branch --set-upstream-to=origin/dev    关联远程仓库

6: git pull
